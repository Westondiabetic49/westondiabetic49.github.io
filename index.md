## 🔍 Fixing Common Errors

Some users see error messages during setup. These errors occur because of Windows security or missing system files.

### Error: Installer Stops Suddenly
This happens when an old version of the software exists on your PC. 

1. Go to your Windows Settings.
2. Select Apps.
3. Look for KiCad in the list.
4. Select Uninstall.
5. Restart your computer.
6. Try to install the new version again.

### Error: Missing DLL Files
This means your system lacks necessary code libraries. 

1. Visit the Microsoft website.
2. Search for Visual C++ Redistributable.
3. Download the version for x64 systems.
4. Run the downloaded file.
5. Restart your computer.
6. Run the KiCad installer one more time.

### Error: Access Denied
You do not have the right permissions to write files to the system.

1. Ensure you log in as the computer Administrator.
2. Disable your antivirus temporarily while you install the software.
3. Turn your antivirus back on after the installation finishes.
